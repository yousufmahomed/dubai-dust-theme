# Dubai Dust — Shopify Theme

**"Luxury Within Reach"**

A custom Shopify theme for Dubai Dust, an Arabian luxury marketplace based in Newcastle, KwaZulu-Natal, South Africa.

## ✦ Features

- **Dark Luxury Aesthetic** — #0a0e1a background with gold (#d4a853) accents
- **Velvet Rope Timer** — Exclusivity countdown for collections
- **VIP Deposit Gate** — R250 refundable deposit for ultra-luxury access
- **Surprise 7% Discount** — Auto-applied at checkout for ultra-luxury items
- **Gift Messages** — Available for ultra-luxury orders
- **16 Collections** — 8 standard + 8 ultra-luxury
- **96 Products** — Cosmetics, Jewellery, Perfume, Electronics, Watches, Handbags
- **Mobile-First** — Responsive design with Arabian-inspired elements

## ✦ Quick Start

```bash
# Install Shopify CLI
npm install -g @shopify/cli @shopify/theme

# Login & push
shopify auth login --store dubaidust.myshopify.com
shopify theme push
```

See [SETUP.md](SETUP.md) for full deployment instructions.
